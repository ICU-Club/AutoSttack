# AutoAttack 自动攻击

> 作者：星梦
- 出处： `https://github.com/ICU-Club`

- 这是一个支持宠物自动攻击敌人，无需手动操作，配置文件自动补充宠物名称，只需填写 ID 即可

## 命令

| 语法 | 权限 | 说明 |
| :-----| ----: | :----: |
| `/atog` | `autoattack.use` | 开启/关闭自动攻击系统 |
| `/atog <宠物名>` | `autoattack.use` | 开启/关闭指定宠物的攻击 |
| `/autobind <宠物名> [弹幕] [伤害] [DebuffID] [BuffID]` | `autoattack.bind` | 绑定宠物攻击配置 |
| `/autocfg <参数名> <值> [宠物名]` | `autoattack.admin` | 修改全局或宠物配置参数 |
| `/autolist` | `autoattack.use` | 列出当前宠物配置 |
| `/reload` | `tshock.cfg.reload` | 重载配置文件 |

## 配置

配置文件路径：`tshock/AutoAttack.json`

```json5
{
  "插件总开关": true,
  "默认弹幕类型": 121,        // 默认弹幕ID（紫红光球）
  "默认基础伤害": 50,
  "默认击退力度": 2.0,
  "默认发射冷却": 60,         // 单位：帧（60帧=1秒）
  "默认飞行速度": 16.0,
  "默认索敌范围": 800.0,      // 单位：像素
  "默认启用武器伤害映射": false,
  "默认武器伤害系数": 1.0,
  "默认敌人DebuffID": 0,
  "默认敌人Debuff持续时间": 300,
  "默认玩家BuffID": 0,
  "默认玩家Buff持续时间": 300,
  "宠物配置表": {
    "18": {
      "宠物名称": "暗影珠",    // 可不填写，重载时自动补充
      "弹幕类型": 121,
      "基础伤害": 50,
      "击退力度": 2.0,
      "发射冷却": 60,
      "飞行速度": 16.0,
      "索敌范围": 800.0,
      "AI参数0": 0.0,
      "AI参数1": 0.0,
      "AI参数2": 0.0,
      "启用武器伤害映射": false,
      "武器伤害系数": 1.0,
      "启用此宠物攻击": true,
      "敌人DebuffID": 0,
      "敌人Debuff持续时间": 300,
      "玩家BuffID": 0,
      "玩家Buff持续时间": 300
    }
    // ... 更多宠物配置
  }
}
```

## 攻击模式说明

| 模式 | 说明 |
|------|------|
| 最近敌人模式 | 默认模式，自动锁定范围内最近的敌对NPC |
| 反击模式 | 玩家受到伤害时触发，优先攻击伤害来源的敌人 |


## 注意事项

- 玩家必须先使用 `/atog` 开启自动攻击系统，宠物才会自动攻击
- 冷却单位为帧，60帧约等于1秒
- 索敌范围单位为像素，推荐范围 600-1200
- 启用武器伤害映射后，宠物伤害会随玩家手持武器变化
- 配置文件中只需填写宠物ID，名称会在重载时自动补充
